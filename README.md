# Инструкция по запуску проекта
  1.  Для начала откройте среду разработки
  2.  Загрузите папку с данным проектом
  3.  Откройте терминал
  4.  Установите cypress
  5.  Откройте cypress

### 4. Установка cypress
```
npm install cypress --save-dev
```
```

### 6. Открытие cypress
```
npx cypress open
```
